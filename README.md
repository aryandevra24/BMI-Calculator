## âœ¨ Features

**BMI Calculation**: Calculates BMI using the standard metric formula: $\text{weight (kg)} / (\text{height (m)})^2$.

**Metric Units**: Accepts input for weight in kilograms (kg) and height in centimeters (cm).

**Instant Feedback**: Displays the calculated BMI score with two decimal places.

**Weight Guide**: Provides the standard BMI classification (Under Weight, Normal Range, Over Weight).

**Lightweight & Fast**: No frameworks or external dependencies, ensuring quick loading and execution.
